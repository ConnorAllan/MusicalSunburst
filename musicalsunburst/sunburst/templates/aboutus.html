About Us

For our final project in CS 205, we have choosen to create, design and implement a program that will display a music database in a graphical way on a web server. To accomplish this, we are utilizing Django(2.1.7) as our framework and D3 Sunburst Charts as our graphical display. The database is hosted via the SQLite3 database that accompanies the Django framework.

https://think.cs.vt.edu/corgis/csv/music/music.html -- Derived dataset based on previous option, appears to be more in line with our size constraints and our goals for content.

For information of how our project was done, visit our github with the code and process.

https://github.com/ConnorAllan/MusicalSunburst
